# Document Types

Common document types in Swiss correspondence.

## Invoice Types

| German | English | Description |
|--------|---------|-------------|
| **Rechnung** | Invoice | Standard bill for goods/services |
| **Mietzinsrechnung** | Rent invoice | Monthly rent bill |
| **Honorarrechnung** | Fee invoice | Professional services |
| **Schlussrechnung** | Final invoice | Project completion bill |

## Statements

| German | English | Description |
|--------|---------|-------------|
| **Kontoauszug** | Account statement | Bank account summary |
| **Kreditkartenabrechnung** | Credit card statement | Monthly card charges |
| **Abrechnung** | Statement/settlement | General accounting |
| **Lohnabrechnung** | Payslip | Salary statement |

## Insurance & Health

| German | English | Description |
|--------|---------|-------------|
| **Kostenvoranschlag** | Cost estimate | Pre-approval for treatment |
| **Leistungsabrechnung** | Benefits statement | Insurance payout details |
| **Prämienrechnung** | Premium invoice | Insurance premium bill |
| **Police** | Policy | Insurance contract |

## Official & Administrative

| German | English | Description |
|--------|---------|-------------|
| **Mitteilung** | Notice | General communication |
| **Bescheid** | Decision/notification | Official ruling |
| **Mahnung** | Reminder/dunning | Payment reminder |
| **Kündigung** | Termination | Contract cancellation |
| **Vertrag** | Contract | Agreement document |
| **Fragebogen** | Questionnaire | Survey/form |

## Property & Rent

| German | English | Description |
|--------|---------|-------------|
| **Nebenkostenabrechnung** | Utility statement | Annual utility reconciliation |
| **Verwaltungswechsel** | Management change | Property manager change notice |
| **Mietvertrag** | Lease agreement | Rental contract |

## Detection Keywords

Look for these terms in document body:

- `Rechnung` / `Invoice` → Invoice type
- `Kontoauszug` / `Statement` → Statement type  
- `Mitteilung` / `Information` → Notice type
- `Kündigung` / `Termination` → Termination notice
- `Mahnung` / `Zahlungserinnerung` → Payment reminder
